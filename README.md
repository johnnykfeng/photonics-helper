This is a streamlit app that helps us do common calculations in photonics, such as unit conversions and basic plots.

## Default units
* wavelength [nm] or [1e-9 m]
* omega [Trad/s] or [1e12 rad/s]
* frequency (f) [GHz] or [1e9 Hz]
* Quality factor (Q) [M]
* Transmission or Loss [dB]
* Power_mW [mW] or [1e-3 W]
* Power_dBm [dBm]
